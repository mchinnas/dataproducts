## Introduction
This is the README file for the course project in the Coursera - Developing Data Products course. The application itself is a **simple temperature converter**.  The application converts the temperature from one unit to another namely Celsius to Fahrenheit and vice versa.

The RStudio shiny application for this course project is [hosted on Rstudio's shiny server](https://mchinnas.shinyapps.io/DataProducts-09/) 

### Dependencies:
This application depends on the following libraries:  
* ```shiny``` - required for the shiny application  
  
### Developer usage:
To execute this application download the ```server.R``` and ```ui.R``` files from this repository and use ```runApp()``` to execute from the same folder.  
  
  
### User documentation:
The documentation for the application user is included on the **Documentation** tab within the application.
